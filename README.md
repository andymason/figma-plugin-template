# Minimal Figma Plugin template

Jumping off point for creating lightweight Figma plugins.

## Requirements

- Node >= 14
- [Watchexec ](https://github.com/watchexec/watchexec)

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

Minifies and remove inline sourcemaps.

```bash
npm build
```
